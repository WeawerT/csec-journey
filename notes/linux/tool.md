holehe — это инструмент OSINT для разведки по электронной почте. С его помощью можно выяснить, на каких сайтах зарегистрирован тот или иной email-адрес
Ключевая особенность: он не отправляет никаких уведомлений на проверяемый адрес
```
# Базовый запуск
holehe test@example.com

# Показать только сайты, где есть аккаунт
holehe test@example.com --only-used

# Вывод в формате CSV (удобно для отчетов)
holehe test@example.com --csv
```
